## Live Site

**[https://bridge-neuroscience.github.io/BRIDGE-Neuroscience.github.io/](https://bridge-neuroscience.github.io/BRIDGE-Neuroscience.github.io/)**

## About BRIDGE

BRIDGE is a GPU-accelerated tractography pipeline for reconstructing white matter fiber pathways from high-resolution 3D synchrotron microscopy images.
